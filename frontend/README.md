# STT Web (Vite + React)

This is the frontend for **STT — Simple Time Tracking**.

## Local dev

```bash
cd frontend
npm install

cp .env.example .env
npm run dev
```

### Config

`frontend/.env`:

```env
VITE_API_BASE_URL=http://localhost:8000
```

## Build

```bash
cd frontend
npm run build
npm run preview
```
